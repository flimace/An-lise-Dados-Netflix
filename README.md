# An-lise-Dados-Netflix
Este projeto demonstra o fluxo do meu primeiro projeto de Engenharia e Análise de Dados. O objetivo foi extrair dados brutos do catálogo da Netflix (via Kaggle), realizar limpeza e tratamento com Python, estruturar um banco de dados relacional no SQL Server e criar visualizações com o PowerBI. 

Primeira etapa: tratamento dos dados.
A identificação e tratamento dos valores nulos foi feito para padronizar o banco de dados e deixá-lo aceitável para a exportação para o SQL Server.
A exportação do arquivo otimizado em .csv (UTF-8) evito erros de codificação do banco.
